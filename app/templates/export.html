匯出到 Apple Health

下載 Apple Health 相容 XML(含所有血壓 + 心跳資料,自動 Correlation 包裝 / 時間錯開 / 含 metadata)。 iPhone 端用 Health Import(App Store 免費)匯入。

📦 匯出範圍
🧪 試水溫:最近 7 天 最近 30 天 最近 90 天 📥 全部資料

範圍從「最新資料日期」倒推 N 天(非從今天倒推,避免資料未更新時撈到空集合)。 建議先用「試水溫:最近 7 天」在自己 iPhone 試一次,確認 sys/dia 配對、 時間軸、metadata 都正確顯示後,再把「全部資料」傳給家人。 試完可在健康 App 中刪掉這 7 天。

A. 匯入到「自己的 iPhone」

  1. iPhone App Store 安裝「Health Import」(免費)
  2. 把下載的 XML 用 AirDrop 傳到 iPhone
  3. iPhone 上點檔案 → 用 Health Import 開啟 → 確認匯入
  4. 打開「健康」App → 瀏覽 → 心臟 → 血壓,確認資料已進入

B. 匯入到「家人的 iPhone」(不同 Apple ID)

XML 含敏感醫療資料,建議用 AirDrop 離線傳輸,不要走 LINE/Email/雲端。

  1. 家人 iPhone:設定 → 一般 → AirDrop → 暫時設為「所有人 10 分鐘」
  2. 家人 iPhone 上先裝好「Health Import」(App Store 免費)
  3. 你 Mac:Finder 找到 bp_export_YYYY-MM-DD.xml → 右鍵「分享」→ AirDrop → 選家人裝置
  4. 家人 iPhone:收到通知 → 接受 → 預設用「檔案」App 接收
  5. 家人 iPhone:檔案 App 找到 XML → 長按 → 分享 → 用 Health Import 開啟 → 確認匯入
  6. 家人 iPhone:打開「健康」App → 瀏覽 → 心臟 → 血壓,確認資料已進入
  7. 記得回去把家人 AirDrop 改回「僅聯絡人」或「關閉」

⚙️ 本系統匯出格式特色

  • Correlation 包裝:收縮 + 舒張綁定為一筆完整血壓(120/80),不會被拆成兩個獨立量
  • 時間錯開:同 session 的 L1 / R1 / L2 / R2 自動錯開 1 分鐘(避免「同秒 4 筆」)
  • Metadata 註記:每筆含 bp_arm(左右手)/ bp_sequence(第幾次)/ bp_period(AM/PM)/ bp_source(資料來源),點開可看註記
注意:每次匯出包含所有歷史資料。Health Import 會比對時間戳去重複,但若你已匯入過部分資料,個別工具可能仍會顯示重複項目,可在健康 App 中手動刪除。
